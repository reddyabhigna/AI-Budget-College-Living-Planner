# AI-Budget-College-Living-Planner
The AI Budget College Living Planner is an intelligent financial management system designed specifically for college students who manage money independently. It collects user income and expense details, categorizes spending into fixed and variable components, and analyzes monthly patterns. Using a regression-based forecasting model, the system predicts future expenses and evaluates potential financial risks before overspending occurs.

By comparing predicted expenses with available income, the system generates personalized budget optimization suggestions and recommends achievable savings targets. The solution helps students improve financial discipline, reduce unnecessary spending, and make data-driven financial decisions. Overall, it demonstrates how artificial intelligence can be practically applied to solve real-world student budgeting challenges.
